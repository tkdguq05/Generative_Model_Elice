# Generative_Model_Elice
### 프로젝트 : __Generative Model을 이용한 범죄자 사진 생성__
* 양재 혁신허브 RNCD에서 진행한 AI 실무자 양성과정에서 진행한 프로젝트와 공부한 것을 정리한 레포지토리입니다.

* 공부한 것 : 
  * Kaggle Titanic EDA
  * Linear Regression, OLS
  * Logistic Regression
  * Auto-Encoder
  * Seq-to-Seq
  * GAN & VAE
  * NNS - DNN,CNN,RNN
  * Reinforcement Learning ...
